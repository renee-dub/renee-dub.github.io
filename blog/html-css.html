Sprint 2 - Tech - What are the best practices associated with using Class and ID Selectors?

What is a .Class Selector?

A 'Class' is an attribute used to define a group of HTML elements in order to apply unique styling and formatting to those elements with CSS. It can be used multiple times on a page.It starts with a dot (.) character.

What is an #ID Selector?

An 'ID' is an attribute used to identify a single element where specific CSS formatting and styling is required. It can only be used once on a page. It starts with a hash (#) character.

Best Practice When Using ID and Class Selectors

  • If you are using the same selector more than once on your webpage, you should use the Class selector.
  • If you are using one unique selector on your webpage, then you should use the ID selector.
  • An ID selector will overule most other selectors.
  • Using a Class selector is preferable.
